### Guide to clone

```
git clone https://github.com/itushh/Medisync.git
```

```
cd Medisync
```

```
cd backend
```

```
python -m venv xenv
```

```
#Linux
source xenv/bin/activate
```

```
#Windows
venv\Scripts\Activate
```

```
pip install -r requirements.txt
```

```
uvicorn app.main:app --reload
```
